# ESP32-S3 DevKitM-1

Primary Microcontroller for the SmartFuseBox Project



## Overview

The ESP32-S3 DevKitM-1 is a compact, powerful development board built around the ESP32-S3-MINI module. It features a dual-core LX7 processor, 16 MB Flash, 8 MB PSRAM, Wi-Fi 2.4 GHz, Bluetooth 5.0 LE, and USB connectivity.



## Key Specifications



### Core Hardware

- Processor: 32-bit LX7 dual-core up to 240 MHz
- Flash: 16 MB
- PSRAM: 8 MB
- Voltage: 3.3 V logic
- Chipset: ESP32-S3
- RAM Technology: DDR (board‑level)


### Wireless Connectivity

- Wi-Fi 2.4 GHz 802.11 b/g/n
- Bluetooth 5.0 LE with mesh support


### USB \& I/O

- Dual USB‑C ports:
- Port 1: Power + USB‑UART (programming/debugging)
- Port 2: Native USB (OTG, device emulation, mass storage)
- Main power connector: 24‑pin header
- GPIO: Standard ESP32‑S3 DevKitC layout (40+ pins depending on variant)


### Antenna Options

- Built‑in PCB antenna (default)
- IPEX connector for external antenna
- Note: Switching to external antenna requires DIY soldering (bridge pads).
